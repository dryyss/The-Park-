Machine à états

Le cycle complet d'une transaction sécurisée, de l'acceptation à la clôture. Rien ne se libère tant que la boucle « expédition prouvée → réception confirmée » n'est pas bouclée. Chaque transition est déclenchée par un événement (preuve validée, statut API transporteur, action membre ou détecteur anti-fraude).
